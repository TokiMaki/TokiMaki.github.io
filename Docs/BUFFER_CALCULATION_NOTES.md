# BUFFER_CALCULATION_NOTES

## Core Rule

- 30레벨 버프는 버프강화 스위칭 세팅 기준 적용 스탯과 스위칭 기준 버프력 증폭을 사용한다.
- 50레벨 1차 각성기는 현재 장착 세팅 기준 적용 스탯과 현재 기준 버프력 증폭을 사용한다.
- 현재 `status`에 이미 반영된 패시브/직접 스탯은 중복으로 더하지 않는다.
- 직접 시전해서 적용 스탯에 추가되는 자버프/오라형 스탯은 현재 적용 스탯에 더한 뒤, 스위칭 시 교체 차이를 차감한다.
- 스위칭 계산은 가능한 한 `현재 적용 스탯 -> 스위칭 적용 스탯` 방식으로 분해한다.
- 캐릭터별 스위칭 고정 오프셋은 사용하지 않는다. 현재 장비·아바타·크리쳐에서 버프강화에 실제 등록된 슬롯만 교체하고, 직접 주스탯과 관련 자버프/패시브 스킬 레벨 차이를 API에서 읽어 계산한다.
- 버프강화 상·하의 아바타의 주스탯 엠블렘 차이는 현재/버프강화 API 응답을 매 조회마다 비교해 동적으로 반영한다.
- 현재 칭호의 `버프력 증폭 2%`는 30레벨 버프 시전 시 스위칭 칭호로 바뀌면 빠진다.
- API 스킬 계수는 정수 표기라 숨은 소수점 오차가 있다. 스탯 증가량은 약 0.1%, 공격력 증가량은 약 0.5%까지 정상 오차로 본다.
- 30레벨 버프의 실제 레벨은 `skill/buff/equip/equipment`의 `skillInfo.option.level`을 사용한다.
- 30레벨 버프 기본값은 `힘/지능 = 150 + 11 × 레벨`, `공격력 = 36 + 1.8 × 레벨`로 계산한 뒤 직업계수를 적용한다.
- 1차 각성기 실제 레벨은 `skill/style`의 50레벨 각성기를 기본 15레벨로 보정하고, 현재 장착 장비/칭호/아바타/크리쳐의 `50레벨 액티브 스킬`, 공통 레벨 범위, 직접 스킬 증가량을 모두 합산한다.
- 1차 각성기 기본값은 30레벨부터 레벨당 92씩 증가하므로 `986 + 92 × (레벨 - 30)`으로 계산한다.
- 패시브 스킬 1레벨당 스탯 증가량은 기존 고정값을 유지한다.

## Final Score Formula

```text
score =
(25000 + totalStat) / 25000
* (3300 + totalAtk) / 3300
* K
* 1.165
```

- 보정상수는 `K = 333`을 사용한다.
