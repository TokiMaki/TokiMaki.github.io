## 세트단계 역산 표본 입력 방식

엑셀 입력표의 `역산 기준` 시트에는 변경 전 기준값을 입력한다.

필수 기준값:

```text
기준 장비점수
기준 서약포인트
기준 레어 결정 개수
기준 유니크 결정 개수
기준 레전더리 결정 개수
기준 에픽 결정 개수
기준 태초 결정 개수
기준 세트단계
기준 세트 최종데미지(%)
기준 기타 전후비
```

`세트단계 표본` 시트에는 각 변경 후 표본을 입력한다.

필수 변경 후 값:

```text
변경 후 장비점수
변경 후 서약포인트
변경 후 세트단계
변경 후 레어 결정 개수
변경 후 유니크 결정 개수
변경 후 레전더리 결정 개수
변경 후 에픽 결정 개수
변경 후 태초 결정 개수
기타 전후비
```

역산 목적:

```text
기준 장비점수과 변경 후 장비점수의 전후비,
서약포인트 변화,
등급별 결정 개수 변화,
세트포인트/세트단계 변화를 같이 보고
해당 세트단계의 최종데미지 환산값을 채운다.
```

### 서약의 가호 포인트 보정 규칙

서약의 가호는 등급별 시작값을 기준으로, 같은 등급 구간 안에서 서약 포인트가 25 올라갈 때마다 아래 값이 추가된다.

| 포인트 증가 | 최종데미지증가(%) | 버프력 | 명성 |
|---:|---:|---:|---:|
| +25 | +0.5 | +70 | +35 |

계산식:

```text
구간 내 단계 수 = floor((현재 서약포인트 - 해당 등급 구간 시작 포인트) / 25)

최종데미지증가 = 등급 시작 최종데미지증가 + 구간 내 단계 수 * 0.5
버프력 = 등급 시작 버프력 + 구간 내 단계 수 * 70
명성 = 등급 시작 명성 + 구간 내 단계 수 * 35
```

예시:

```text
레전더리 구간 시작값:
최종데미지증가 39

현재 서약포인트 1700:
구간 내 +25 단계 4회
최종데미지증가 39 + 0.5 * 4 = 41
버프력 +280
명성 +140
```

주의:

```text
이 보정은 각 등급 구간 안에서만 적용한다.
구간이 바뀌면 이전 구간 보정을 누적하지 않고, 새 등급 시작값에서 다시 계산한다.

예: 2100점이 다음 구간의 시작점이면
최종데미지증가 50, 버프력 8210, 명성 4105를 그대로 사용한다.
```

## 입력 규칙

- 퍼센트 값은 `%` 기호 없이 숫자만 입력한다.
- 공식 API 값이 `391.1%`라면 `391.1`로 입력한다.
- 같은 세트단계에서 세트명이 달라도 값이 같으면 표본 캐릭터/세트명만 추가 기록한다.
- 세트단계별 값이 확정되면 장비점수 계산에서는 세트 설명문을 해석하지 않고 이 표의 최종데미지증가 값을 사용한다.
