# DNF_HELL_OPTIMIZER_SPEC.md

## 1. 문서 목적

이 문서는 던전앤파이터 천해천/조율자 헬 운영 최적화 사이트의 상세 설계서다.

Codex가 조율자 계산, 헬 기대값, 정가, 초월, 결정 조각, 솔리드 소울, 계정 단위 추천 로직을 수정할 때 이 문서를 기준으로 삼는다.

이 사이트는 단순한 "헬 vs 정가 비용 계산기"가 아니다.

핵심 목표는 다음 질문에 답하는 것이다.

```text
현재 상태에서 더 헬을 도는 것,
정가를 치는 것,
초월을 쓰는 것 중
무엇이 계정 전체 졸업 기대값을 가장 많이 줄이는가?
```

## 2. 사이트 목표

이 사이트는 던전앤파이터 천해천/조율자 시즌의 헬 운영, 정가, 초월, 한정 재화 사용을 판단하기 위한 계정 단위 운영 도구다.

사이트가 제공해야 하는 핵심 판단은 다음과 같다.

- 현재 계시로 몇 캐릭까지 헬을 돌릴 수 있는가
- 어떤 캐릭을 우선 헬 돌리는 게 좋은가
- 특정 캐릭은 더 헬을 돌아야 하는가, 마감 재화를 기다려야 하는가
- 정가가 가능한 세트가 있을 때 실제로 정가를 치는 게 이득인가
- 솔리드 소울을 어느 캐릭에게 쓰는 것이 계정 전체 효율이 가장 좋은가
- 특정 행동이 졸업 기대비용을 얼마나 줄이는가

## 3. 사이트에서 하지 않을 것

인게임에서 이미 확인 가능한 콘텐츠 완료 체크리스트는 만들지 않는다.

예를 들어 다음 기능은 우선순위가 낮다.

- 캐릭별 주간 콘텐츠 완료 체크
- 일일 숙제 완료 체크
- 던전별 입장 여부 관리
- 단순 출석 체크형 기능

이 사이트의 가치는 체크리스트가 아니라, 계정 전체 자원 배분 판단에서 나온다.

## 4. 졸업 기준

기본 졸업 기준은 다음과 같다.

- 유효 세트 기준 3 태초 + 8 에픽

서약 결정을 장착할 수 있는 칸은 11칸이고 태초 장비는 3개까지만 착용 가능하므로, 실질적인 졸업 목표는  3 태초 + 8 에픽이다.

따라서 계산 로직은 특정 세트 또는 유효 세트 루트가 다음 조건에 도달하는지를 기준으로 판단한다.

- 태초 개수 >= 3
- 에픽 이상 개수 >= 8
- 그중 태초는 최대 3개만 실질 가치 반영.

실무적으로는 다음과 같이 처리한다.

- 졸업 충족 = 태초 3개 이상 + 에픽 8개 이상

서약 결정은 중복 허용으로 같은거 밖에 뜨지 않고 그냥 장착이 가능하다. 같은 세트 3태초 + 8에픽을 졸업 기준으로 본다.

## 5. 핵심 자원 모델
## 5.1 계시

계시는 헬을 더 돌 수 있는 연료다.

계시는 다음 계산에 직접 연결된다.

- 추가 가능 헬 판수 = 보유 계시 / 헬 1판 순소모

기본값:

- 집 기준 헬 1판 입장비 = 49계시
- PC방 or 아라드 패스 지정 기준 헬 1판 입장비 = 44 계시
- PC방 and 아라드 패스 지정 기준 헬 1판 입장비 = 39 계시
- 최신 표 기준 헬 1판 회수율 = 14.60 계시

단, 회수율 기준에 따라 순소모 값은 바뀔 수 있어야 한다.

## 5.2 광휘

광휘는 정가와 초월에 모두 들어가는 핵심 마감 재화다.

기본 환산값:

- 광휘 1개 = 계시 40

광휘 가치는 고정이다.

## 5.3 에픽 소울

저울과 에픽 정가에 사용되는 재화다.

기본 환산값:

- 에픽 소울 1개 = 계시 90

## 5.4 태초 소울

에픽 저울과 태초 정가에 사용되는 재화다.

기본 환산값:

- 태초 소울 1개 = 계시 1000

## 5.5 결정 조각

결정 조각은 세트별로 쌓이는 정가 선택권이다.

조각이 충분하다는 것은 정가 가능 조건을 만족했다는 뜻이지, 정가가 무조건 이득이라는 뜻은 아니다.

정가 가능 여부와 정가 실행 가치는 반드시 분리한다.

- 조각 = 정가 가능성을 여는 선택권
- 정가 실행 여부 = 정가 후 졸업 기대비용 감소량과 재화 기회비용으로 판단

## 5.6 솔리드 소울

솔리드 소울은 계정 단위 한정 초월권이다.

고정 계시 가치만으로 판단하면 안 된다.

솔리드 소울은 계정 전체에서 가장 큰 졸업 기대비용 절감처에 우선 사용해야 한다.

## 6. 정가와 초월 비용
## 6.1 에픽 정가

에픽 정가 = 해당 세트 결정 조각 1000 + 광휘 200 + 에픽 소울 20

계시 환산 기본값:

- 광휘 200 × 40 = 8000
- 에픽 소울 20 × 90 = 1800
- 합계 = 9800 계시 가치

결정 조각은 해당 세트 정가 선택권으로 따로 판단한다.

## 6.2 태초 정가

태초 정가 = 해당 세트 결정 조각 1500 + 광휘 1000 + 태초 소울 5

계시 환산 기본값:
- 광휘 1000 × 40 = 40000
- 태초 소울 5 × 1000 = 5000
- 합계 = 45000 계시 가치

결정 조각은 해당 세트 정가 선택권으로 따로 판단한다.

## 6.1 에픽 초월

에픽 초월 = 솔리드 소울 150 + 광휘 200

계시 환산 기본값:

- 광휘 200 × 40 = 8000
- 합계 = 8000 계시 가치

솔리드 소울은 계정 단위 한정 재화이므로 별도 기회비용으로 판단한다.

## 6.3 태초 초월

태초 초월 = 광휘 500 + 솔리드 소울 500

광휘 환산 기본값:
- 광휘 500 × 40 = 20000 계시 가치

솔리드 소울은 계정 단위 한정 재화이므로 별도 기회비용으로 판단한다.

## 7. 결정 조각 구조

조율자에서 서약결정이 드랍될 때는 등급과 세트가 정해진 결정이 드랍된다.

그 결정을 해체하면 해당 결정의 세트에 대응하는 결정 조각이 오른다.

예시:

칠흑 유니크 서약결정 드랍
→ 해체
→ 칠흑 결정 조각 +12
무리 태초 서약결정 드랍
→ 해체
→ 무리 결정 조각 +100

단, 레어 서약결정은 예외다.

레어 서약결정은 모든 세트 조각을 각각 +1 올려준다.

## 8. 특정 세트 기준 결정 조각 기대값

특정 세트 기준 판당 결정 조각 기대값은 다음 공식으로 계산한다.

특정 세트 기준 판당 조각 기대값
= 레어확률 × 1
+ 유니크확률 × 12 / 12
+ 레전더리확률 × 28 / 12
+ 에픽확률 × 50 / 12
+ 태초확률 × 100 / 12

여기서 /12는 조각이 해체 후 랜덤 세트에 들어간다는 뜻이 아니다.

유니크 이상 서약결정은 결정에 붙은 해당 세트의 조각만 올려준다.

다만 특정 세트 하나만 기준으로 볼 때, 드랍되는 결정의 세트가 12개 중 해당 세트일 기대확률을 반영해야 하므로 /12를 적용한다.

레어는 모든 세트에 +1씩 올라가므로 /12하지 않는다.

현재 기준 특정 세트 판당 결정 조각 기대값은 약 다음 값으로 본다.

특정 세트 기준 판당 결정 조각 기대값 ≈ 0.788개

## 9. 정가 가능과 정가 이득은 다르다

정가 가능 여부와 정가 실행 가치는 반드시 분리한다.

예시 1:

- 태초 0 / 에픽 4 / 조각 1500

이 경우 태초 정가는 가능하지만, 졸업까지 아직 멀기 때문에 정가 가치가 낮을 수 있다.

예시 2:

- 태초 2 / 에픽 8 / 조각 1500

이 경우 태초 정가 1회로 3 태초 + 8 에픽에 도달할 수 있으므로 정가 가치가 매우 높다.

따라서 정가 판단은 다음 순서로 한다.

1. 해당 세트가 졸업 후보인지 확인
2. 정가 후 졸업 조건에 얼마나 가까워지는지 계산
3. 정가 전 졸업 기대비용과 정가 후 졸업 기대비용을 비교
4. 광휘/소울/조각 기회비용을 반영
5. 순절감이 충분히 크면 정가 후보로 표시

## 10. 헬 기대비용 모델

헬 기대비용은 현재 상태에서 헬만 계속 돌렸을 때 졸업까지 필요한 기대 계시를 의미한다.

기본 구조:

헬 졸업 기대비용
= 졸업까지 필요한 기대 판수 × 헬 1판 순소모

여기서 기대 판수는 다음 요소에 영향을 받는다.

현재 세트별 태초 개수
현재 세트별 에픽 개수
유효 세트 루트 개수
에픽/태초 드랍률
해당 드랍이 유효픽일 확률
조각으로 인한 정가 가능성
이미 보유한 정가/초월 재화

초기 버전에서는 복잡한 완전 확률 모델보다, 다음 기준으로 단순화할 수 있다.

현재 세트 상태별 남은 필요 개수
× 해당 등급 유효 획득 기대판수
× 헬 1판 순소모

단, 계산 함수는 나중에 몬테카를로나 분위수 모델로 교체할 수 있도록 분리한다.

## 11. 분위수 모델

사용자는 평균 기대값만으로는 체감과 맞지 않을 수 있다.

따라서 졸업 기대비용 계산에는 분위수 값을 둘 수 있다.

기본 분위수:

66%

의미:

이 정도 판수 안에 목표를 달성할 확률이 약 66%인 지점

UI에서는 사용자가 분위수를 바꿀 수 있어야 한다.

예시:

50% = 중앙값
66% = 기본 추천값
80% = 보수적 목표
90% = 매우 보수적 목표

분위수 값이 바뀌면 헬 기대비용과 정가/초월 판단이 즉시 갱신되어야 한다.

## 12. 마감 행동 가치 판단식

정가/초월 같은 마감 행동의 가치는 다음 방식으로 판단한다.

마감 행동 가치
= 현재 헬 졸업 기대비용
- 행동 후 남은 헬 졸업 기대비용
- 행동에 쓰는 재화의 기회비용
- 조각/솔리드 같은 한정 재화의 기회비용

실무 표현:

이 행동을 하면 졸업 기대비용이 얼마나 줄어드는가?
그 절감폭이 사용한 재화의 기회비용보다 큰가?
이 한정 재화를 다른 캐릭터에게 쓰는 것보다 나은가?

## 13. 에픽 정가 판단

에픽 정가는 다음 조건을 모두 평가한다.

1. 해당 세트 조각이 1000개 이상인가
2. 해당 세트가 유효 졸업 후보인가
3. 에픽 1개 정가 후 졸업 기대비용이 얼마나 줄어드는가
4. 광휘 200 + 에픽 소울 20의 기회비용보다 절감폭이 큰가
5. 같은 재화를 다른 캐릭터에게 쓰는 것보다 우선순위가 높은가

에픽 정가가 가능한데 추천하지 않는 경우도 있어야 한다.

예시 문구:

정가 가능하지만 추천 낮음:
태초 0 / 에픽 4
존버 권장

## 14. 태초 정가 판단

태초 정가는 재화 소모가 크므로 더 엄격하게 판단한다.

판단 조건:

1. 해당 세트 조각이 1500개 이상인가
2. 해당 세트가 유효 졸업 후보인가
3. 태초 1개 정가 후 졸업 조건에 크게 가까워지는가
4. 특히 2태초 + 8에픽 상태처럼 즉시 졸업 또는 준졸업이 가능한가
5. 광휘 1000 + 태초 소울 5의 기회비용보다 절감폭이 큰가

추천도가 높은 예시:

태초 2 / 에픽 8 / 조각 1500
→ 태초 정가 1회로 졸업 조건 충족
→ 정가 우선 후보

추천도가 낮은 예시:

태초 0 / 에픽 4 / 조각 1500
→ 태초 정가는 가능하지만 졸업 기대비용 절감폭이 낮음
→ 정가 보류

## 16. 에픽/태초 초월 판단

에픽/태초 초월은 솔리드 소울을 사용하므로 계정 단위로 판단한다.

판단 조건:

1. 초월 대상 에픽/태초가 실제 유효 세트/유효 부위인가
2. 초월 후 해당 캐릭의 졸업 기대비용이 얼마나 줄어드는가
3. 광휘 200/500의 기회비용을 반영했는가
4. 솔리드 소울 150/500을 다른 캐릭에게 쓰는 것보다 우선순위가 높은가
5. 초월 후 즉시 졸업 또는 큰 폭의 기대비용 감소가 발생하는가

초월 후보는 캐릭터별 순절감 기대값 기준으로 정렬한다.

예시:

솔리드 사용 후보

1. 마키로그
- 초월 전 졸업 기대비용: 72,000 계시
- 초월 후 졸업 기대비용: 18,000 계시
- 광휘 기회비용: 20,000 계시
- 순절감: 34,000 계시 - 솔리드 가치

2. 마키듀얼
- 순절감: 21,000 계시 - 솔리드 가치

## 17. 사용자 입력 최소화 원칙

세트별 결정 조각을 전부 유저에게 입력시키면 사용성이 크게 떨어진다.

기본 흐름은 다음과 같이 설계한다.

1. 사이트가 먼저 정가 후보 세트를 찾는다.
2. 정가 후보로 뜬 세트만 유저에게 확인시킨다.
3. 확인 방식은 숫자로 입력하는데, 실 게임에서 몇개가 있는지 태초 기준의 퍼센트로 보여주게 된다. 그 퍼센트를 입력하게 만든다. ex) 1500개면 100%
4. 세트별 조각 숫자 직접 입력은 고급 설정에 둔다.

## 18. 판단 결과 단계

결과는 단순히 헬/정가로 나누지 않는다.

추천 판정 단계:

판정	        의미
헬 유지	        유효 세트 루트가 많고 헬 기대효율이 좋음
마감 대기	    정가/초월 후보지만 조각/소울/광휘 등 재화 조건 확인 필요
정가/초월 우선	 재화 사용 시 졸업 기대비용 절감이 큼
헬 중단	        헬 기대효율이 낮고 마감 재화 대기 권장

## 21. 마감 후보 화면

마감 후보 화면은 다음 그룹으로 나눈다.

1. 정가 즉시 추천
2. 정가 가능하지만 보류
3. 조각 확인 필요
4. 솔리드 초월 후보
5. 헬 중단 후보

각 후보에는 다음 정보를 표시한다.

- 캐릭터명
- 후보 세트
- 현재 세트 상태
- 행동 전 졸업 기대비용
- 행동 후 졸업 기대비용
- 재화 기회비용
- 순절감
- 추천/보류 사유

## 26. 열린 질문

아직 확정이 필요한 부분은 다음과 같다.

- 졸업 기대값을 정확히 어떤 확률 모델로 계산할 것인가
- 단순 기대값 기반으로 시작할지, 분위수/몬테카를로 기반으로 갈지
- 조각 부족 상태를 "예상 추가 헬 판수"로 얼마나 적극 반영할지
- 솔리드 소울의 기본 기회비용을 숫자로 둘지, 후보 간 순위 비교만 할지
- 세트별 유효 부위/중복 부위 처리를 어떤 입력 구조로 받을지

초기 버전에서는 단순 기대값 기반으로 시작하고, 계산 함수 구조만 나중에 고도화 가능하게 분리한다.

## 27. 최종 요약

이 사이트의 방향은 "던파 조율자판 환산 주스텟 사이트"다.

핵심 자원 해석은 다음과 같다.

계시 = 헬 시행권
조각 = 세트별 정가 선택권
솔리드 = 계정 한정 초월권
졸업 기대값 = 모든 판단의 기준선

사이트가 최종적으로 해야 하는 일은 다음과 같다.

현재 상태에서 더 헬을 도는 것,
정가를 치는 것,
초월을 쓰는 것 중
무엇이 계정 전체 졸업 기대값을 가장 많이 줄이는지 보여준다.
